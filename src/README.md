# gBooks-js

Simple vanilla JavaScript app with Webpack, Bable and SASS support.

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```
Server will be loaded on port 8080


### Build Prod Version

```
npm run build
```
Prod files available in /dist folder
